# LORCA — Long-term Rates of Carbon Accumulation

This folder will contain workflows for C-14 radiocarbon dating of sediment
cores, used to build long-term (centennial to millennial scale) age-depth
chronologies.

## Planned content

- Bayesian age-depth modelling with **Bacon** (`rbacon`)
- Calibration of radiocarbon dates with **Bchron** or **OxCal**
- Integration of Pb-210 (RERCA) and C-14 chronologies
- Long-term carbon accumulation rate calculations
